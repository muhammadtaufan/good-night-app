# GoodNight App
Simple Rails API to track sleep time and wake up time

## Getting Started
### Prerequisites
```
Ruby version 3.0.0
Rails version 7.0.3
```

### Installing

- Clone the repo
```
git clone https://github.com/muhammadtaufan/good-night-app.git
```
- Install dependencies & Migration

```
bundle install
cp config/application.yml.sample config/application.yml
rails db:migrate
```

### Usage
Run the server
```
rails s
```

### Run the tests
```
rspec
```
